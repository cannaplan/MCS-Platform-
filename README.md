# MCS Platform

The MCS Platform is the digital operating system for organizing supporters, volunteers, county chapters, partner organizations, and future statewide initiatives.

## Objectives

* Support statewide organizing efforts
* Manage volunteers and events
* Track initiative progress
* Coordinate county chapters
* Build partnerships with schools, tribal nations, businesses, and community organizations

## Technology Stack

* Next.js
* TypeScript
* PostgreSQL
* Prisma
* Tailwind CSS
* Auth.js

## Roadmap

### MCS 1.0

* User accounts
* Volunteer management
* County chapters
* Events
* Signature tracking
* Admin dashboard

### MCS 2.0

* School registry
* Tribal partnerships
* Project management
* Research library

### MCS 3.0

* Greenhouse operations
* Food distribution
* Renewable energy monitoring
* Workforce development
